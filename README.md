# Project Setup

> Create `.env` file

> Paste your **`DATABASE URI`**
```
  DATABASE_URI=<your_database_uri>
```

> Set your **`JWT SECRET KEY`**
```
  SUPER_SECRET_KEY=<your_jwt_secret_key>
```

> Set your **`INTERFACE`** and **`PORT`**
```
  INTERFACE_PORT=<IP>:<PORT>
```

> Run your project
```
  cargo run --release
```
